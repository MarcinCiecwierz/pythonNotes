
# Notes

This project is set to store notes in db and show it in simple frontend via streamlit.

Project can be connected to LLM chats via mcp servers locally. Just add *mcp/server.py* file to LLM and it automatically will know how to talk with app.

# Requirements

To start project you need python.

After cloning add .env file with **API_URL** where url is url to backend.

to run project simply run .sh file

`
./run.sh
`
